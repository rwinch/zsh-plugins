= cursor-rules

A zsh plugin that provides a command to link your Cursor rules into the current directory. This ensures that `$PWD/.cursor/rules` is a symlink to your shared rules.

== Overview

When you run the `cursor-rules` command, the plugin will:

* Ensure `$PWD/.cursor/rules` exists as a symlink to `~/code/rwinch/.cursor/rules`.

This is useful for explicitly sharing common AI coding rules into your local project directories.

== Installation

With antigen:

[source,bash]
----
antigen bundle rwinch/zsh-plugins/cursor-rules
----

Or source the plugin directory so that `cursor-rules.plugin.zsh` is loaded by your framework.

== Usage

Run the following command while inside a directory where you want your cursor rules applied:

[source,bash]
----
cursor-rules
----

This will create or update the `.cursor/rules` symlink in the current directory pointing to `~/code/rwinch/.cursor/rules`.

== Requirements

* Zsh
* The directory `~/code/rwinch/.cursor/rules` should exist; the plugin will display an error if it does not.
